## 项目主要需求

### 项目概述

【Web前端】科研数据可视化分析工具。纯前端科研数据分析与图表绘制平台，支持 CSV/JSON 数据导入、多维数据筛选、图表配置和分析报告生成。数据存储于 localStorage / IndexedDB。**本项目用于验证模型跨文件协同改动能力，禁止将多个职责合并进单一文件。**

### 核心实体（≥ 3 个数据模型）

1. **数据集（Dataset）** — 字段：数据集名称、描述、来源文件名、导入时间、行数、列数、列定义数组（列名+数据类型Number/String/Date/Boolean）、标签列表、数据JSON
   - 定义位置：`types/dataset.ts`

2. **图表配置（ChartConfig）** — 字段：图表ID、关联数据集ID、图表名称、图表类型（Line/Bar/Pie/Scatter/Heatmap/Boxplot/Radar）、X轴字段、Y轴字段、分组字段、颜色方案、标题、副标题、图例位置、自定义样式JSON
   - 定义位置：`types/chart.ts`

3. **分析报告（Report）** — 字段：报告ID、报告名称、创建时间、更新时间、描述、包含的图表ID列表、布局配置（行列排列）、导出状态
   - 定义位置：`types/report.ts`

4. **数据筛选器（Filter）** — 字段：筛选器ID、关联数据集ID、字段名、操作符（Equals/Contains/GreaterThan/LessThan/Between/In）、值、是否激活
   - 定义位置：`types/filter.ts`

5. **统计摘要（StatSummary）** — 字段：关联数据集ID、字段名、均值、中位数、标准差、最小值、最大值、分位数、空值数
   - 定义位置：`types/stat.ts`

### 核心页面（≥ 4 个）

1. **数据工作台** `/workspace` — 消费：Dataset + Filter
   - 数据集列表，导入CSV/JSON，数据表格预览（支持列排序、筛选），数据类型自动识别和手动修正
   - 使用 `<DataGrid>`, `<FilterPanel>`, `<EmptyState>` 共享组件

2. **图表编辑器** `/chart-editor` — 消费：Dataset + ChartConfig
   - 左侧选数据集和字段，中间图表预览（实时更新），右侧配置面板（图表类型、轴映射、颜色、样式）
   - 使用 `<ChartPreview>`, `<FieldSelector>`, `<ColorPicker>`

3. **报告组装** `/report-builder` — 消费：Report + ChartConfig
   - 拖拽排列图表到报告画布，调整布局，添加标题和说明文字，预览报告
   - 使用 `<ChartPreview>`, `<DraggableGrid>`

4. **统计分析** `/statistics` — 消费：Dataset + StatSummary
   - 选择数据集和字段，自动计算统计摘要，相关性矩阵，数据分布直方图
   - 使用 `<StatCard>`, `<DataGrid>`

5. **图表库** `/charts` — 消费：ChartConfig
   - 已保存的图表列表（缩略图预览），支持搜索和标签筛选，点击进入编辑
   - 使用 `<ChartPreview>`, `<EmptyState>`

### 横切关注点

1. **全局异常处理**
   - 触达文件层：文件解析异常（CSV格式错误）+ 图表渲染异常 + IndexedDB异常 + ErrorBoundary + 错误类型定义

2. **主题切换（深色/浅色）**
   - 触达文件层：CSS变量 + themeStore + 组件适配 + 图表主题色切换 + 主题类型定义

### 共享枚举/常量（≥ 2 组）

1. **图表类型 ChartType**：Line / Bar / Pie / Scatter / Heatmap / Boxplot / Radar
   - 前端：`types/enums.ts`

2. **数据类型 DataType**：Number / String / Date / Boolean

3. **筛选操作符 FilterOperator**：Equals / Contains / GreaterThan / LessThan / Between / In

4. **颜色方案 ColorScheme**：Default / Pastel / Vivid / Monochrome / Scientific

### 共享前端组件清单（≥ 3 个）

1. **`<ChartPreview>`** — 图表编辑器、报告组装、图表库
2. **`<DataGrid>`** — 数据工作台、统计分析
3. **`<FieldSelector>`** — 图表编辑器、统计分析（字段选择下拉）
4. **`<FilterPanel>`** — 数据工作台
5. **`<StatCard>`** — 统计分析
6. **`<EmptyState>`** — 所有空状态

### 共享 hooks/utils 清单（≥ 2 个）

1. **`useDataImport`** — 封装CSV/JSON文件解析（PapaParse + JSON.parse），数据类型推断
2. **`useChartConfig`** — 封装图表配置的创建、更新、验证逻辑
3. **`useLocalStorage`** — 通用本地存储hook
4. **`statistics/`** — 统计计算工具（均值、标准差、分位数、相关系数）

### 技术要求

- **前端端口**：18803
- **前端**：React 18 + TypeScript + Vite
- **图表**：ECharts 5（高级图表配置）
- **UI 库**：Tailwind CSS
- **表格**：AG Grid Community
- **CSV解析**：PapaParse
- **拖拽**：@dnd-kit/core
- **状态管理**：Zustand
- **本地存储**：IndexedDB（Dexie.js）
- **路由**：React Router v6
- **构建工具**：Vite
